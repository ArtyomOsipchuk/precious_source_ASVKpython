Техническая документация¶

Initialization file.

class chessclub.client.ChessCmd(username)¶
do_createtable(arg)¶

Создать новый стол для игры. Использование: createtable [as white|black] Если цвет не указан, выбирается случайным образом. Можно создать только один стол одновременно (до leave).

do_join(arg)¶

Присоединиться к существующему столу. Использование: join [id] Без аргументов — быстрое присоединение к свободному столу. Можно присоединиться только к одному столу одновременно (до leave).

do_leave(arg)¶

Покинуть текущий стол (выйти из партии/лобби). Использование: leave После выхода можно создать или присоединиться к другому столу.

do_list(arg)¶

Показать список всех столов, их игроков и статус. Использование: list

do_play(arg)¶

Начать игру, если оба игрока присоединились к столу и готовы. Использование: play Окно игры откроется и терминал будет заблокирован до окончания партии или выхода.

do_quit(arg)¶

Завершить работу клиента. Использование: quit Перед выходом автоматически покидает текущий стол.

do_switchlocale(arg)¶

Сменить локаль.

do_view(arg)¶

Посмотреть партию за выбранным столом в режиме зрителя. Использование: view <id>
